# AI Microservice Project

This repository contains a multi-module Spring Boot project with:

- API Gateway Service (JWT, rate limiting, routing)
- AI Processing Service (Spring AI, PDF extraction, PostgreSQL, Redis)
- Docker setup for local development
- Architecture diagrams and documentation

## Modules

- `gateway-service/`
- `ai-service/`
- `docker/`
- `docs/`

## Tech Stack

- Java 21
- Spring Boot 3.x
- Spring Cloud Gateway
- Spring AI
- Redis
- PostgreSQL
- Docker & Docker Compose
